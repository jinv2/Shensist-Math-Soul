# Shensist Math Soul - 神思数学灵魂 (Math Gear 2026)

> “在齿轮转动的一瞬，数理的逻辑化作了自然的呼吸。”

**Shensist Math Soul** 是一项面向 2026 年人工智能进校园标准的教育实验。它不仅是一个数学工具，更是一个具有“灵魂”的 AI 智能体 (AI Agent)，通过 3D 机械结构与山海经神兽叙事，将乏味的算术运算转化为一场探索真理的对话。

## 🪐 核心理念 (Philosophy)

- **非灌输式学习 (Non-Prescriptive)**：取消分值、对错与计时，鼓励孩子自主拨动齿轮，探索数字之间的关系。
- **灵机交互 (Spiritual Interaction)**：每一次计算都会触发神兽显现、文字共鸣与色彩变幻，建立数学与情感的连接。
- **2026 AI 标准**：专为未来教育体制设计，将 AI 视为陪伴者与引导者，而非简单的答题机器。

## ✨ 核心特性 (Features)

1. **3D 机械齿轮 (3D Mechanics)**：使用 WebGL 与 CSS 3D 渲染的两个 0-9 机械滚轮，具备真实的物理惯性与机械声音。
2. **万象发现录 (Spirit Scroll)**：实时记录探索历史，以古代卷轴的形式展现每一次数理发现。
3. **四神兽映射 (Spirit Beasts)**：
   - **加法 (+)** - 🐉 龙：生机与增长
   - **减法 (-)** - 🐢 玄武：厚重与退守
   - **乘法 (×)** - 🔥 凤：爆发与聚合
   - **除法 (÷)** - 🦄 麟：秩序与分割
4. **动态色彩情感 (Dynamic Emotion)**：界面颜色随运算符实时切换，营造沉浸式的元素氛围。
5. **智能体灵魂 (AI Soul)**：内置 `soul.js` 逻辑引擎，具备会话记忆，不仅提供计算结果，更提供创意提问。

## 🛠️ 技术布局 (Tech Stack)

- **Vanilla JS**: 核心物理引擎与粒子系统。
- **Tailwind CSS**: 响应式界面与现代美学布局。
- **Web Audio API**: 合成的机械滚动声学反馈。
- **CSS 3D Transforms**: 沉浸式机械结构渲染。
